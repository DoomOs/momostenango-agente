Crud.py:

Funciones CRUD para las tablas principales: - ciudadanos - sesiones - consultas_respuestas - documentos - faqs

Incluye saneamiento básico y validación de datos.

async app.db.crud.crear_ciudadano(nombre: str, email: str, telefono: str | None = None) → int

Crea un nuevo ciudadano.

Parámetros:

  • nombre – Nombre completo.

  • email – Correo electrónico único.

  • telefono – Teléfono opcional.

Devuelve:

ID del ciudadano creado.

async app.db.crud.crear_sesion(ciudadano_id: int, token_sesion: str) → int

Crea una nueva sesión para un ciudadano.

Parámetros:

  • ciudadano_id – ID del ciudadano.

  • token_sesion – Token único de sesión.

Devuelve:

ID de la sesión creada.

async app.db.crud.guardar_consulta_respuesta(sesion_id: int, pregunta: str, respuesta: str, confianza: float)

Guarda una consulta y su respuesta en la base de datos.

Parámetros:

  • sesion_id – ID de la sesión.

  • pregunta – Texto de la pregunta.

  • respuesta – Texto de la respuesta.

  • confianza – Nivel de confianza del agente (0 a 1).

async app.db.crud.obtener_ciudadano_por_email(email: str) → dict | None

Busca un ciudadano por su correo electrónico.

Parámetros:

email – Correo electrónico.

Devuelve:

Diccionario con datos del ciudadano o None si no existe.

async app.db.crud.obtener_faqs(limit: int = 10) → List[dict]

Obtiene una lista de preguntas frecuentes.

Parámetros:

limit – Número máximo de FAQs a obtener.

Devuelve:

Lista de diccionarios con preguntas y respuestas.

async app.db.crud.obtener_sesion_por_token(token_sesion: str) → dict | None

Obtiene la sesión por token.

Parámetros:

token_sesion – Token de sesión.

Devuelve:

Diccionario con datos de la sesión o None si no existe.
